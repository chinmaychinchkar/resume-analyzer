 AI Resume Analyzer

This project analyzes resumes using NLP and Machine Learning to extract skills and match them with job roles.



 Features
- Resume parsing (PDF)
- Skill extraction using spaCy
- Job matching using TF-IDF & Cosine Similarity
- Real job dataset (CSV)
- Streamlit web application
- Confidence score and explanation

 Tech Stack
- Python
- spaCy
- Scikit-learn
- Pandas
- Streamlit
- PyPDF2


How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the app:
streamlit run app.py





 Project Description

Developed an AI-based Resume Analyzer using NLP (spaCy) and Machine Learning (TF-IDF, Cosine Similarity) to extract skills and intelligently match job roles. Includes skill boosting and explanation system for better predictions.
